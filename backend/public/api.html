🚀 API Endpoints

POST /api/register Register a new user
POST /api/login Login user
GET /api/events Get all events
GET /api/events/{id} Get event details
POST /api/events Create event (Admin)
POST /api/events/{id}/register Register for event
GET /api/registrations Get user registrations

🔐 Authentication

This API uses Bearer Token authentication. Include the token in the Authorization header:

Authorization: Bearer YOUR_TOKEN_HERE

💡 Getting Started

  1. Register a new user at POST /api/register
  2. Login to get an access token at POST /api/login
  3. Use the token to access protected endpoints
  4. Explore the full documentation in Swagger UI
